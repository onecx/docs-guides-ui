== Module Federation and Preloaders

*Preloaders* are lightweight applications that load before the shell application. They define and provide packages for *Module Federation sharing*.

=== Execution Flow
[mermaid]
----
graph LR
  gatherPreloader[Gather Preloaders] --> loadPreloader[Load Preloaders with Module Federation]
  loadPreloader --> getSharedPackages[Get shared packages]
  getSharedPackages --> waitForCompletion[Bootstrap Application]
----

* Gather the list of preloaders before starting the application
* Schedule the load of all preloaders with *Module Federation*
* Wait for completion
* Once all are complete, start bootstrapping the application
