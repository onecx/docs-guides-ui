== Module Federation and Preloaders

*Preloaders* are lightweight applications that load before the shell application. They define and provide packages for *Module Federation sharing*.

=== Execution Flow
Before *shell initialization*, shell is:

  * gathering list of preloaders to load
  * scheduling load of all preloaders by using *module federation* and waits on the completion
  * once all complete, starts *bootstrapping the app*
