[#configuration-service-runtime-configuration]
= ConfigurationService and environment variables (runtime configuration)

[#configuration-service-configure-env-json]
== Configure `env.json`

`ConfigurationService` loads runtime configuration from `assets/env.json`.
If `remoteConfigURL` is set, that URL is used instead.

`src/assets/env.json` is created in the Shell application.
The file must contain a JSON object with string key/value pairs.

.Example `src/assets/env.json`
[source,json]
----
{
  "APP_BASE_HREF": "${APP_BASE_HREF}",
  "KEYCLOAK_REALM": "${KEYCLOAK_REALM}",
  "KEYCLOAK_URL": "${KEYCLOAK_URL}",
  "KEYCLOAK_CLIENT_ID": "${KEYCLOAK_CLIENT_ID}",
  "KEYCLOAK_TIME_SKEW": "${KEYCLOAK_TIME_SKEW}",
  "KEYCLOAK_UPDATE_TOKEN_MIN_VALIDITY": "${KEYCLOAK_UPDATE_TOKEN_MIN_VALIDITY}",
  "KEYCLOAK_ON_TOKEN_EXPIRED_ENABLED": "${KEYCLOAK_ON_TOKEN_EXPIRED_ENABLED}",
  "KEYCLOAK_ON_AUTH_REFRESH_ERROR_ENABLED": "${KEYCLOAK_ON_AUTH_REFRESH_ERROR_ENABLED}",
  "APP_VERSION": "${APP_VERSION}",

  "IS_SHELL": true,

  "AUTH_SERVICE": "${AUTH_SERVICE}",
  "AUTH_SERVICE_CUSTOM_URL": "${AUTH_SERVICE_CUSTOM_URL}",
  "AUTH_SERVICE_CUSTOM_MODULE_NAME": "${AUTH_SERVICE_CUSTOM_MODULE_NAME}",
  "AUTH_SERVICE_CUSTOM_BFF_URL": "${AUTH_SERVICE_CUSTOM_BFF_URL}",

  "ONECX_PORTAL_SEARCH_BUTTONS_REVERSED": "${ONECX_PORTAL_SEARCH_BUTTONS_REVERSED}",
  "POLYFILL_SCOPE_MODE": "${POLYFILL_SCOPE_MODE}"
}
----

Placeholders may be replaced during deployment.

[#configuration-service-access-variables]
== Access variables

Values are available after `ConfigurationService.init()` completes.

[source,ts]
----
import { Component, inject } from '@angular/core';
import { CONFIG_KEY, ConfigurationService } from '@onecx/angular-integration-interface';

@Component({
  selector: 'my-shell-component',
  template: `...`,
})
export class MyShellComponent {
  private readonly configurationService = inject(ConfigurationService);

  async ngOnInit() {
    const keycloakUrl = await this.configurationService.getProperty(CONFIG_KEY.KEYCLOAK_URL);
    const allValues = await this.configurationService.getConfig();

    this.configurationService.config$.subscribe((currentConfig) => {
      console.log(keycloakUrl, allValues, currentConfig);
    });
  }
}
----

Common access patterns:

- `await configurationService.getProperty(CONFIG_KEY.KEYCLOAK_URL)`
- `await configurationService.getConfig()`
- `configurationService.config$.subscribe(...)`
